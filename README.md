Rime 官方提供的五笔 86 输入法的配置并不会四码自动上屏（无重码时），
需要在 .schema.yaml 配置文件里的 speller 段落下添加 max_code_length: 4 和 auto_select: true 这两行就可以了。这是一个小坑，貌似只在官方更新日志里提及了。其他的在官方手册里讲得很详细。很赞。
